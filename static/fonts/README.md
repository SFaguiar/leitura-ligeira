# OpenDyslexic local assets

The reader ships `OpenDyslexic-Regular.woff2` and `OpenDyslexic-Bold.woff2`
locally. They are used only after a reader explicitly chooses OpenDyslexic in
its profile preferences; the system font remains the default and fallback.

- Upstream package: `@fontsource/opendyslexic` version `5.3.0`
- License: SIL Open Font License 1.1, included verbatim in
  `OpenDyslexic-OFL-1.1.txt`
- Regular SHA-256:
  `F007004AF3CDA5D8076E57C943F8CC8D00A0DA25988B1AE1048683D60E7CAC1A`
- Bold SHA-256:
  `DD9FA9C7991113B0DDDEFE9506A30AD26B48E302B7A8FB91719A4726E8FDE85B`

No font is downloaded from a CDN at runtime. The feature is a visual
preference, not a medical treatment or a claim of universal reading benefit.
