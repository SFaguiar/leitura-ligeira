# Certificados locais

Esta pasta recebe o certificado e a chave privada opcionais do HTTPS local.
Somente este README é versionado; os arquivos de certificado e, principalmente,
a chave privada permanecem ignorados pelo Git.

Os nomes detectados automaticamente pelo inicializador são:

- `leitura-ligeira.pem`
- `leitura-ligeira-key.pem`

Consulte a seção "HTTPS local opcional" do README principal para gerar o par
com mkcert e instalar a autoridade local nos dispositivos que precisarem
acessar o servidor sem aviso do navegador.
